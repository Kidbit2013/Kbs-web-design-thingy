1. Gather Parts and Tools

Collect your CPU, motherboard, RAM, storage, power supply, case, cooler, and a screwdriver. Anti-static protection is essential.

2. Install the CPU and Cooler

Place the processor into the socket carefully, latch it in, then attach the cooler with thermal paste for good heat transfer.

3. Add Memory and Storage

Install RAM modules in the correct slots, then mount SSDs or HDDs. Secure cables cleanly for airflow.

4. Mount the Motherboard

Secure the motherboard inside the case using standoffs and screws, then connect the case headers and power cables.

5. Connect the Graphics Card

Insert the GPU into the PCIe slot and attach any required power cables. Keep cables tidy to avoid obstructing fans.

6. Test and Boot

Power on the system and enter the BIOS to verify temperatures, storage, and RAM. Install your operating system once the hardware check is complete.

Build safely, choose compatible parts, and enjoy a powerful custom PC.
